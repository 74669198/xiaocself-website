Claude Code 快捷键与隐藏操作大全

写在前面 用 Claude Code 一个月，我发现操作速度的差距不在打字快慢，而在知不知道这些快捷键。 这篇把 Claude Code 的所有快捷操作整理成一张速查表，建议收藏。 一、斜杠命令（/ commands） 在对话中输入 / 触发： 命令 作用 使用场景 /clear 清空当前会话上下文 话题切换时，避免上下文干扰 /compact 压缩对话，保留关键信息 对话长了（20+轮）但不想重启 /cost 查看当前会话 token 消耗 担心费用超标时 /help 显示所有可用命令 忘了命令时 /model 切换 AI 模型 需要换更强/更快的模型 /exit 或 Ctrl+D 退出 Claude Code 结束工作 最常用的是 /compact。当对话超过 20 轮，Claude 开始"失忆"或回复变慢时，输入 /compact 让它自动总结前面的内容，然后继续。比重启会话更省时间。 二、文件引用技巧 @ 符号引用 输入 @ 弹出文件选择器，支持： 输入 效果 @filename 引用单个文件 @src/**/*.tsx 引用整个目录（glob 模式） @README.md 引用项目文档 @package.json 引用配置文件 为什么重要：Claude 的上下文窗口有限，精准引用文件 = 节省 token = 更准确的回答。 ...
