我的日常工具箱#

这个页面记录我每天都在用的工具，不是"听说好用"的收藏夹。每个工具都附带了真实使用时长、月成本和一句话评价。

AI 核心工具#

工具用途使用频率月成本评价
Claude Code代码、数据处理、自动化每天 3-5 小时免费目前最强大的命令行 AI，Agent 能力远超 Cursor
Kimi (K2.5)长文本、报告撰写每天 1-2 小时~￥30200K 上下文是真的能用上，读整本书都没问题
Notion AI笔记整理、会议纪要每天 30 分钟￥80和 Notion 结合得最好，但单独买不太值
Midjourney配图、海报素材每周 2-3 次$10 (~￥70)v7 的审美依然领先，但中文理解还是差

组合成本：约 ￥180/月

替代方案（如果预算有限）：

  • Claude Code → 用 CC-Switch 接入 Kimi（免费）
  • Notion AI → 直接用 Kimi 整理后粘贴
  • Midjourney → Leonardo.ai（免费额度够用）

最低成本版：只用 Kimi API + Claude Code，月成本约 ￥30

开发工具#

工具用途用了多久评价

Warp终端1 年AI 命令提示是真有用，尤其是记不清参数时
TablePlus数据库管理2 年比 Navicat 轻，比 DBeaver 好看

最近想试的：Zed 编辑器（据说快得离谱）

效率工具#

工具用途用了多久评价
Raycast启动器 + 快捷操作2 年macOS 必装，窗口管理、剪贴板历史、计算器都在里面
CleanShot X截图 + 录屏1 年截图后标注、马赛克、OCR 一条龙，写文档离不开
Espanso文本替换6 个月把常用短语变成快捷键，比如 ;addr 自动展开地址
Kap录屏转 GIF3 年免费开源，录操作演示够用

一个细节：Raycast 的剪贴板历史我设置了 200 条，经常发现三天前复制的代码还在，意外救命。

工具用途用了多久评价
飞书文档团队协作、知识库2 年比 Notion 快，比语雀好看，国内团队首选

弃用的工具#

不是不好，是不适合我：

工具弃用原因替代
Cursor太重，启动慢，不如 Claude Code 轻量Claude Code
ChatGPT Plus网络不稳定，响应慢Kimi
Typora收费后寻找替代VS Code + Markdown 插件
AlfredRaycast 免费且更好看Raycast

工具选择原则#

  1. 先免费后付费：确认真的需要再花钱
  2. 能命令行就不 GUI：快，且能自动化
  3. 能开源就不商业：避免被绑架
  4. 定期清理：每季度检查一次，不用就删
