CC-Switch 配置指南：接入 Kimi/GLM 等国内模型

为什么要用 CC-Switch？ Claude Code 默认连接 Anthropic 官方 API，在国内访问可能遇到： 网络不稳定 响应速度慢 偶尔无法连接 CC-Switch 是解决这个问题的神器——它让 Claude Code 可以无缝切换到国内大模型（Kimi、GLM、通义千问等），速度快、稳定性高。 前置准备 Claude Code 已安装（见入门指南） 有火山引擎/智谱AI/阿里云的 API Key 安装 CC-Switch # 全局安装 npm install -g cc-switch # 验证安装 cc-switch --version 配置 Kimi 模型（推荐） 1. 获取 API Key 登录 火山引擎 → 创建接入点 → 复制 API Key 2. 配置环境变量 # 添加到 ~/.zshrc 或 ~/.bashrc export ANTHROPIC_BASE_URL="https://ark.cn-beijing.volces.com/api/coding" export ANTHROPIC_AUTH_TOKEN="你的API-Key" export ANTHROPIC_MODEL="Kimi-K2.5" 3. 应用配置 source ~/.zshrc # 验证 claude 如果看到 Claude Code 正常启动，并且回复内容来自 Kimi，说明配置成功！ ...
