## 写在前面

用 Claude Code 一个月，我发现**操作速度的差距不在打字快慢，而在知不知道这些快捷键**。

这篇把 Claude Code 的所有快捷操作整理成一张速查表，建议收藏。

---

## 一、斜杠命令（/ commands）

在对话中输入 `/` 触发：

| 命令 | 作用 | 使用场景 |
|------|------|---------|
| `/clear` | 清空当前会话上下文 | 话题切换时，避免上下文干扰 |
| `/compact` | 压缩对话，保留关键信息 | 对话长了（20+轮）但不想重启 |
| `/cost` | 查看当前会话 token 消耗 | 担心费用超标时 |
| `/help` | 显示所有可用命令 | 忘了命令时 |
| `/model` | 切换 AI 模型 | 需要换更强/更快的模型 |
| `/exit` 或 `Ctrl+D` | 退出 Claude Code | 结束工作 |

**最常用的是 `/compact`**。当对话超过 20 轮，Claude 开始"失忆"或回复变慢时，输入 `/compact` 让它自动总结前面的内容，然后继续。比重启会话更省时间。

## 二、文件引用技巧

### @ 符号引用

输入 `@` 弹出文件选择器，支持：

| 输入 | 效果 |
|------|------|
| `@filename` | 引用单个文件 |
| `@src/**/*.tsx` | 引用整个目录（glob 模式） |
| `@README.md` | 引用项目文档 |
| `@package.json` | 引用配置文件 |

**为什么重要**：Claude 的上下文窗口有限，精准引用文件 = 节省 token = 更准确的回答。

### Tab 补全

输入文件名的一部分按 `Tab`，自动补全路径：

```
@src/comp[Tab] → @src/components/
```

## 三、终端快捷键

Claude Code 运行在终端里，所有 readline 快捷键都适用：

| 快捷键 | 作用 |
|--------|------|
| `Ctrl + C` | 取消当前输入或中断运行中的命令 |
| `Ctrl + L` | 清屏（只重绘，不删除输入内容） |
| `Ctrl + A` | 光标移到行首 |
| `Ctrl + E` | 光标移到行尾 |
| `Ctrl + U` | 删除光标到行首的内容 |
| `Ctrl + K` | 删除光标到行尾的内容 |
| `Ctrl + W` | 删除光标前一个单词 |
| `↑` / `↓` | 浏览历史命令 |
| `Ctrl + R` | 搜索历史命令 |

**注意 `Ctrl + L`**：之前版本有 bug 会清空输入框，最新版（2.1.126）已修复，现在只清屏不影响输入。

## 四、多行输入技巧

### 方式一：Shift + Enter

按 `Shift + Enter` 换行，不发送消息：

```
帮我写一个函数，要求：[Shift+Enter]
1. 接收一个数组 [Shift+Enter]
2. 返回去重后的结果 [Enter发送]
```

### 方式二：Markdown 代码块

用三个反引号包裹代码，Claude 会自动识别：

`````
```python
def hello():
    print("world")
```
请帮我优化这段代码
`````

### 方式三：粘贴大段内容

直接粘贴不会误发送，Claude 会等你的后续指令。

## 五、权限操作快捷键

Claude 执行危险操作前会提示确认：

| 操作 | 你的选项 |
|------|---------|
| `Y` | 允许一次（仅本次） |
| `A` | 允许全部（当前类别所有操作） |
| `N` | 拒绝 |
| `Ctrl + C` | 取消 |

**建议**：
- 批量操作时按 `A`，省得反复确认
- 不确定的操作按 `N`，让 Claude 解释清楚再决定

## 六、隐藏技巧

### 1. 让 Claude "只读"模式

如果你只想问问题、看代码，不想让它修改任何文件：

```bash
claude --read-only
```

或对话中说：
```
接下来我只问问题，不要修改任何文件
```

### 2. 指定目录启动

```bash
claude /path/to/project
```

直接 cd 到项目目录启动，省一步操作。

### 3. 从剪贴板粘贴图片

新版支持直接粘贴图片（`Cmd + V`），Claude 会分析图片内容。超过 2000px 会自动压缩，不会崩会话。

### 4. 查看当前配置

```
/config
```

查看当前生效的所有 settings.json 配置。

### 5. 快速切换模型

```
/model
```

弹出模型选择器。如果你配置了网关（如 CC-Switch），会从网关的 `/v1/models` 自动拉取可用模型列表。

## 七、效率组合技

### 组合 1：快速审查大量文件

```
@src 目录下所有文件，帮我检查是否有未使用的导入
```

Claude 会自动遍历并给出清单。

### 组合 2：批量重构

```
把所有 console.log 替换成 logger.debug，只改 src 目录下的 .ts 文件
```

Claude 会搜索、确认、批量替换。

### 组合 3：上下文保留切换

```
/compact
继续刚才的话题，帮我生成单元测试
```

压缩后不丢失关键信息，继续深入。
